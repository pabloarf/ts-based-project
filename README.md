## Initial setup

Initial commands for initializing the repo:

```bash
npm init -y

npm install typescript ts-node @types/node --save-dev

# Install ESLint
npm install eslint --save-dev

npx eslint --init

# Setup Testing
npm install jest @types/jest ts-jest --save-dev

npx ts-jest config:init
```
